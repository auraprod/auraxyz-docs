## Global Functions

### `GetDatamodel()`

Returns the root `RobloxInstance` representing the Roblox DataModel. This is the entry point for accessing all other instances in the game.

**Returns:** A `RobloxInstance` object representing the DataModel, or `nil` if the DataModel address is not available.
